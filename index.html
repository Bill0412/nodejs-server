This is the documentation for our ARM API

To use the api, following are samples: (You can try it out in Shell)

curl -d '{"signal":"1"}' -H "Content-Type: application/json" -X POST 35.185.172.243:2222

curl -d '{"signal":"Hello"}' -H "Content-Type: application/json" -X POST 35.185.172.243:2222

As shown above, you only need to modify the signal attribute in the JSON text.
